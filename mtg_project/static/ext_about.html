About MtG Guru

Our website is a Magic:the Gathering database. We have 12 different magic sets with all the cards from each availeable for viewing. We have a cards page which displays 2 randomly generated cards per set (24 total). Each of these links to a dedicated card page which shows the card art and all of the card information in an easy to see and understand way. We have a types page which displays all of the different types of magic cards, and each one goes to a page which describes what types are and a list of the various subtypes per. We have a how to play page that provides a breif description and 5 different videos. The last page we have is the Api-implementation page. Our group had the NBADB page. We decided to get all of their teams and add their hometowns to a google map and when you click on the marker you will be taken to their page for that particular team.

Technologies used: Twitter bootstrap, Django, mySQL, Apiary, Google Maps API, Python, Javascript, JQuery, HTML5, CSS3
